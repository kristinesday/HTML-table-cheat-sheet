HTML Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<th> Table Head data Table head row container.
<tr> Table row The table row container.
<td> Table data The table row container - data in each cell.
<tfoot> Table Foot The set of rows defining the footer in a table.
<colspan> Column Span Defines how many columns a td element should span or combine in one cell.
<rowspan> Row Span Defines how many rows a td element should span and combine in one cell.

How to create a table in HTML?

  1. Add <table> tags. <table> for opening and </table> for closing.
  2. Inside of them add <thead>, <tbody> and <tfoot>. Remember the closing tags.
  3. <thead> is the top aka Main row which typically contain information of what will go in each column.
  4. To define how many rows the Table Head will contain, add <tr> and </tr> elements in the <thead> tags to define it. It is typically one row. However, this row normally contains multiple cells that later on grows into columns.
  5. To define what information each Table Head cell will contain, add <th> elements. By adding text in-between the <th> and </th> you can define column content e.g. Tags, Name of the tags and description. Depending on how many columns you wan to add, add more of <th> and </th>.

    6. <thead>
    <tr>
    <th>Tag<th>
    <th>Name<th>
    <th>Description<th>
    </tr>
    </thead>

  6. To add more rows below Table head, use <tr> and </tr>. Inside of them place <td> and </td> for Table data e.g. specific name of the tag and description how to use it. Remember to add as many <td> as necessary to follow the collumn count. In this case 3.
  7. If you want to combine some of the collumn on row cells into one, use <colspan> or <rowspan>
  8. <colspan> will spread element in multiple collumns. <rowspan> will spread element in multiple rows. To define in how many collumns or rows, use <td colspan="2"> element inside the tag
